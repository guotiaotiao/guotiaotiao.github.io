An analogy to describe the differences between HTML and CSS

Using the analogy of building a house, html is the basic structure of a house. css is for decoration. A house without decoration is a house, it just doesn't look good. And it is JS that decides whether the house is a hospital, a cinema, or a residence. JS makes it functional and interactive.


Explain control flow and loops using an example process from everyday life

Use bubble tea as an analogy to Control flow and Loop. The original order should be, boil water, put the tea in the cup, brew the tea, add milk, stir it, and drink the tea. This process is the control flow. But there is also a possibility that when adding milk, it is found that the milk is bad, but the bad milk has been added to the tea, so the tea has to be poured out and a new bottle of milk is opened. Then take another cup , put the tea in it, brew the tea, add milk, stir, and drink it. The process of putting tea in cup, brewing tea and adding milk is a loop.
